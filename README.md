# transformada-fourier-analisis-senales
Simulación y análisis de señales mediante la Transformadora de Fourier usando Python.

Este repositorio contiene el código en Python desarrollado para la simulación y análisis
de señales en el dominio del tiempo y de la frecuencia utilizando la Transformada de Fourier.

El código fue elaborado con fines académicos. Algunas librerías utilizadas (NumPy y Matplotlib)
no son compatibles con todos los entornos de ejecución, por lo que el análisis gráfico se
documenta directamente en el informe del proyecto.
